# CSS Injector

A browser extension that allows you to inject custom CSS styles into specific websites.

## What is it?

CSS Injector lets you create domain-specific CSS rules that are automatically applied to matching websites as you browse. Perfect for customizing the appearance of your favorite sites, fixing layout issues, or applying your own visual preferences across the web.

## Key Features

- Create custom CSS rules for specific domains
- Enable/disable rules individually
- Import/export your CSS configurations
- Persistent settings across browser sessions
- Real-time CSS injection on page load

## Installation

1. Clone this repository
2. Open your browser's extension management page
3. Enable "Developer mode"
4. Load the `src` directory as an unpacked extension

## Usage

Click the extension icon and navigate to Settings to create your first CSS rule. Each rule requires a name, domain pattern, and your custom CSS code.
